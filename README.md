# :red_car: Sistema de Cadastro de Veículos - Condomínio

### Sobre o projeto
Sistema web para **cadastro e gerenciamento** de veículos de moradores de um condomínio. A aplicação permite cadastrar, listar, atualizar e excluir veículos, facilitando o controle e a gestão de acesso no ambiente comdominal.

### Objetivo do projeto
Projeto faz parte do meu processo de aprendizado em desenvolvimento web, com foco em boas práticas, organização e entendimento completo do fluxo entre frontend, backend, banco de dados e estruturação de rotas (GET, POST, PUT, DELETE).

### Funcionalidades 
* Login de usuários (porteiro / síndico);
* Cadastro de veículos;
* Listagem de veículos;
* Atualizações de informações;
* Exclusão de cadastros e login;
* Comunicação via API com banco de dados;

### Tecnologias
* JavaScript
* Node.js
* APIs REST
* HTML5 / CSS3
* Bootstrap
* JSON
* Banco de Dados (MySQL)
